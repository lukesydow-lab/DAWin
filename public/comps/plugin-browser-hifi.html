Plugin Browser — Hi-fi · Studio FX Rack

The hi-fi version of the inline plugin browser. The FX chain becomes a wooden cabinet holding rack units; each plugin is a piece of outboard gear with brushed-metal faceplate, screws, LCD readout, and a glowing power LED. Owner color appears as trim along the cabinet rails.
The structure is the lo-fi. The difference is texture, depth, and material. Every interaction (click power, hover for grip, drag to reorder, click "+" to open browser) is identical — only the visual treatment changes. The browser popover itself stays clean and functional rather than going full rack-aesthetic, because the browser is a tool for finding plugins, not a piece of gear.

The Cabinet

Wooden top + bottom rails (dark walnut gradient with subtle vertical grain striations).

Owner-color trim runs as a 2px stripe with glow along the inside edge of each rail. This is the only place owner color appears in the FX chain — strong identity signal without overpowering the gear.

Top rail shows track name + slot count. Bottom rail shows interaction hint.

The Rack Unit

Brackets (left/right) — gunmetal vertical strips with two Phillips screws each. Standard rack-ear convention.

Faceplate — dark anodized aluminum with brushed-metal scanlines. Slight gradient (lighter top, darker bottom) suggests subtle curvature.

Plugin name in embossed silver caps + subtitle line ("Channel Strip · 4-Band") for character.

LCD readout — black inset with amber LED-style monospace text. Shows current param values. Dims to ~18% when unit is bypassed.

Power LED

Recessed well with a faux-LED dome inside. Off = dark grey. On = bright core in owner color with halo glow.

Inner highlight at the top of the dome simulates the lens reflection — gives the LED a tactile 3D quality.

Try toggling it: hover any unit and click the power button. Watch the LCD dim with it.

Drag to reorder

Cursor changes to grab over the unit; grabbing on press. Unit lifts -1px on hover with deeper shadow (already wired).

In production: drag a unit by the bracket area; a purple insertion line glows between rack units to show drop target. Order matters in a signal chain.

Empty slot

The "+ Add Unit" footer is now a diagonal-striped empty rack slot. Reads as a real empty slot in the cabinet waiting for a unit to be inserted.

Hover lights up in owner color with a glow halo — clearly tappable. Click opens the same plugin browser popover.

The Browser (popover)

Same content as the lo-fi: search → Recent → Favorites → categories → "Browse all plugins…" footer.

Slight visual upgrade: brushed-metal gradient on the search header bar, owner-color focus ring on the search input.

Stays clean and functional — the browser is a finding tool, not a piece of gear. Resisting the urge to over-design here is intentional.
