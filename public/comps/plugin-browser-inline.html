Anchor & placement

Popover anchors to the "+ Add Plugin" button at the bottom of the existing PluginChainPanel.

Opens to the left of the chain panel (away from the screen edge). Width 340px, max-height 480px.

The arranger remains fully visible — collaboration cues, clip drops, playhead, and other channels all stay in view.

Information hierarchy

1. Search at top — autofocused on open. Power users never leave the keyboard.

2. Recent — last 2–3 used plugins. Most adds repeat recent choices.

3. Favorites — user-pinned plugins.

4. By category — EQ, Dynamics, Reverb/Delay, Modulation, etc.

Interaction model

• Click "+ Add Plugin" → popover scales in from bottom-right with a soft spring (240ms)

• Type to filter — categories collapse to match results

• ↑↓ nav, Enter to add, Esc closes

• Click outside the popover → closes

• Clicking a plugin adds it to the chain, popover closes, new card slides in at the bottom with a brief confirmation pulse

Power button + drag handle

Power button (the round dot on each card) replaces the slidey toggle. Off = dark, on = purple core with halo. Reads as a hardware power button — matches studio-rack convention.

Drag grip appears on hover at the left edge of each card. Grab to reorder the chain; a purple drop-line appears between cards to show the insertion point. Order matters in a signal chain — this needs to be obvious.

Try it now: hover any card to see the grip dots appear; click any power button to toggle.

Future: hi-fi as studio FX rack

The lo-fi above is the structure. When this goes to hi-fi the plugin cards become rack units: brushed metal faceplates, recessed power indicator LEDs, drop-shadow depth, screw heads at corners, mini parameter readouts in a black inset panel.

The chain becomes a vertical stack of rack units in a wooden cabinet — owner color appears as the cabinet trim. Audible "click" sound when a unit is enabled/disabled (optional).

Same interaction model, premium aesthetic.

The escape hatch

"Browse all plugins…" in the footer opens a richer modal for deep discovery: full descriptions, screenshots, presets, "try before you add" preview.

This is the secondary path — most adds happen inline. Modal is a future spec.
