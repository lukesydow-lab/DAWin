Style Crossfade

The renderer interpolates between style passes based on continuous zoom value:

1× → macro envelope (gradient body)
3× → normal RMS + peaks
8× → sticks crossfading in
24× → oscillation line crossfading in
80× → microscopic clean mode (no glow)

Crossfades use a smooth 2-stop opacity blend so neither style pops in or out.
